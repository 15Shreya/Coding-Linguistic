# Min XOR value

> Given an integer array A of N integers, find the pair of integers in the array which have minimum XOR value. Report the minimum XOR value.

### Sample Input
 Input 1:
    
    A = [0, 2, 5, 7]
    
Input 2:

    A = [0, 4, 7, 9]


### Sample Output

Example Output 1:

    2
Explanation:

    0 xor 2 = 2
    
Output 2:
    
    3
    
### Constraints:

2 <= N <= 10<sup>5</sup>  
0 <= A[i] <= 10<sup>9</sup>
